## 项目简介

AIAC2025_v3 是一个自动化量化因子挖掘工具，用于 WorldQuant BRAIN 平台。主要功能：

- **数据探索**：浏览平台数据集、字段、操作符
- **Alpha 构建**：手动或模板化生成 Alpha 表达式
- **批量回测**：并发执行多 Alpha 回测，支持断点续传
- **AI 增强**：智能分析数据特征，自动生成 Alpha
- **错误自愈**：自动诊断并修复失败的 Alpha

## 安装配置

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置凭证

在 `config/` 目录下创建以下文件：

**credentials.json** - BRAIN 平台登录凭证：
```json
["your_email@example.com", "your_password"]
```

**api_keys.json** - AI 服务配置（可选）：
```json
{
    "current_provider": "deepseek",
    "deepseek": {
        "api_key": "your_api_key",
        "base_url": "https://api.siliconflow.cn/v1",
        "model": "Pro/deepseek-ai/DeepSeek-V3.2"
    }
}
```

## Alpha Factory Pipeline

全自动 Alpha 挖掘流程，推荐使用。

### 工作流程

```
AI 数据分析 → 模板调度 → Alpha 生成 → 去重 → 聚类 → 批量回测 → 筛选 → 错误自愈
```

### 使用方法

```bash
# 基础用法
python main.py pipeline --region USA --datasets pv1

# 多数据集
python main.py pipeline --region USA --datasets pv1,analyst15

# 使用针对性模板（推荐）
python main.py pipeline --region USA --datasets analyst4 --template-mode specialized

# 指定输出目录
python main.py pipeline --region USA --datasets pv1 --output-dir research/my_study

# 禁用错误自愈
python main.py pipeline --region USA --datasets pv1 --no-self-heal
```

### 参数说明

| 参数 | 简写 | 说明 |
|------|------|------|
| `--region` | `-r` | 区域代码：USA/CHN/IND/EUR/ASI/GLB/JPN/KOR/TWN |
| `--datasets` | `-d` | 数据集 ID，多个用逗号分隔 |
| `--output-dir` | `-o` | 输出目录路径 |
| `--template-mode` | - | `default`（通用模板）或 `specialized`（针对性模板） |
| `--no-self-heal` | - | 禁用错误自愈功能 |

### 输出报告

Pipeline 结束后生成 `research_report.json`：

```json
{
  "templates_used": 25,
  "alpha_generated": 150,
  "after_dedup": 80,
  "backtest_success": 75,
  "high_quality_count": 12
}
```

## 常见问题

### Q: 登录失败？

1. 检查 `config/credentials.json` 格式是否正确
2. 确认邮箱密码无误
3. 检查网络连接

### Q: 回测报错？

常见错误类型：
- **字段不存在**：检查字段名拼写，使用菜单 3 查看可用字段
- **类型错误**：向量字段需用 `vec_avg()` 聚合后再用 `rank()`/`ts_*`
- **语法错误**：检查括号匹配、操作符参数

### Q: 如何清除缓存？

删除 `cache/` 目录下的 JSON 文件，或在交互模式选择"强制刷新缓存"。

### Q: AI 功能不可用？

1. 确认已创建 `config/api_keys.json`
2. 检查 API Key 是否有效
3. 确认网络可访问 AI 服务
